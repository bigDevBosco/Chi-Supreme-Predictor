How To Use

Input all the values. Input average goals scored, average goals conceded by a particular team. Don't mix it up, make sure you don't input any data for away in the home field and vice versa. If you looking for where to get data visit Footystat

Any match you want to predict, open each team on a new tab. You will see the required data with recent form included. Type recent form from the latest(right-most) to the last. It is five in number, input w for win, d for draw and l for lose To reduce error make sure to input every data in its right field

Understanding The Result
  1. Possible Correct Score is the result gotten after comparing the goals scored and conceded by both teams
  2. Possible Outcomes is based on the possible correct score.
  3. Random Choice is a random selection from the possible outcome
  4. Winning Prediction is the actual 1X2 prediction and it will not work without the recent form. Somethings the result may contradict with possible correct score that is because of recent inform is not used in the possible correct score. The prediction accuracy is included, 75% and 100% are the best.
